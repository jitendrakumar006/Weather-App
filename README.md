# 🌦️ Weather App

A simple Weather Application built using Python with a graphical user interface created using Tkinter
This app allows users to check the current weather of any city using the OpenWeather API.

## 🚀 Features of project
- Search weather by city name
- Shows temperature in Celsius
- Displays weather condition
- Shows humidity
- Simple Tkinter UI

## 🛠 Technologies Used
- Python
- Tkinter
- Requests
- OpenWeather API

## 📦 Installation

1. Clone the repository

git clone https://github.com/jitendrakumar006/weather-app.git

2. Go to project folder

cd weather-app

3. Install dependencies

pip install -r requirements.txt

4. Run the app

python weather_app.py

## 🔑 API Setup

1. Create account at https://openweathermap.org
2. Generate API key
3. Replace in code:

api_key = "YOUR_API_KEY"
